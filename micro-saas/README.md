# ReviewReply

AI-powered Google Review response tool for agencies and multi-location brands.

**Pricing:** $49/mo Starter | $99/mo Growth | $199/mo Agency  
**Stack:** FastAPI + Stripe + OpenAI + Google OAuth

## Quick Start
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Deploy
```bash
npx wrangler pages deploy . --project-name reviewreply
```
