# Ghost Systems — Client Onboarding Checklist

## Pre-Launch (Day -3 to 0)
- [ ] Contract signed + setup fee received
- [ ] Client intake form completed:
  - [ ] Company name, address, service areas
  - [ ] Primary/secondary technician names + phones
  - [ ] Calendar system (Google / Outlook / Calendly URL)
  - [ ] Business hours, after-hours emergency policy
  - [ ] Common services + pricing (diagnostic, tune-up, install)
  - [ ] Competitor differentiation (what makes them unique)
- [ ] Vapi phone number purchased + forwarded from client main line
- [ ] Aria system prompt customized with client branding
- [ ] Calendar integration tested (3 test bookings)
- [ ] SMS alerts tested to all technician phones
- [ ] Backend dashboard credentials sent to client

## Go-Live (Day 1)
- [ ] Client announces AI receptionist to staff (no surprises)
- [ ] Main business line forwarded to Vapi number
- [ ] Monitor first 10 calls in real time
- [ ] Adjust prompt based on edge cases
- [ ] Send client "Go-Live" email with dashboard link

## Week 1
- [ ] Daily check: bookings, missed calls, transcript reviews
- [ ] Client call: feedback, any adjustments needed
- [ ] Update FAQ based on new caller questions

## Month 1
- [ ] Weekly analytics emails sent every Monday
- [ ] Mid-month pulse check call
- [ ] End-of-month report: calls, bookings, ROI calculation
- [ ] Guarantee check: did we book 3+ jobs? If not, apply free month

## Month 2+
- [ ] Monthly report auto-generated
- [ ] Quarterly business review (upsell opportunities: SMS marketing, review generation)
